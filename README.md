### change_mac.sh

Change l'adresse MAC d'une interface réseau.

**Utilisation :**
```bash
sudo bash change_mac.sh
```

- Liste les interfaces réseau disponibles
- Choix entre MAC aléatoire ou manuelle
- Le changement est temporaire (perdu au redémarrage)
